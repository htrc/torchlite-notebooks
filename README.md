# torchlite-notebooks
A collection of Jupyter Notebooks demonstrating Torchlite.

## Prerequisites
- Python 3.11
- Poetry 1.4

## Installation
``` sh
 cd torchlite-notebooks
 poetry install
 ```

## Running
```sh
cd torchlite-notebooks
poetry run jupyter notebook
```

Select the notebook you'd like to use from the menu that appears.
